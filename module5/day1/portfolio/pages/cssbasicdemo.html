8. Box Model

The CSS Box Model consists of:

  • Content: The actual content
  • Padding: Space inside the border
  • Border: Line surrounding the padding
  • Margin: Space outside the border
Margin → Border → Padding → Content

Quick Reference

  • color: Sets text color
  • background-color: Sets background color
  • font-size: Sets text size
  • font-weight: Sets text boldness (bold, normal)
  • font-style: Sets text style (italic, normal)
  • text-align: Aligns text (left, center, right, justify)
  • padding: Inner spacing
  • margin: Outer spacing
  • border: Element border (solid, dashed, dotted, double, inset, outset)
  • border-radius: Rounded corners
  • display: Display type (block, inline, inline-block)
  • border inset: Creates a pressed/sunken effect
  • border outset: Creates a raised/popped-out effect
